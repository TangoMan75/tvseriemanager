\ControllerScraperController

Back-office Scraper controller.

Scrapes imdb TV series and episodes, sends result to DefaulManager for database storage.

This class is not meant for front end purposes.

Routes :

../scrape/
../scrapepages/[:from]/[:to]
../scrapeserie/[:title]

Offers methods : 
scrapeMostPopularSeries()
scrapePages($from, $to)
scrapeSerie($title)
